## 项目结构说明

---

#### framework             **公共模块框架**（包括拦截器、身份验证、请求代码，公共工具类等等）

#### server                         **业务**（包括controller、entity、service、mapper、utils）

#### start		  				**启动项**（包括启动类、配置文件）

---

## 各模块依赖

**server**依赖**framework**

**start**依赖**server**和**framework**
